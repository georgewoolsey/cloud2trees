# tools for working with projections, crs, epsg, and whatnot

tools for working with projections, crs, epsg, and whatnot

## Usage

``` r
get_vertical_crs(x)
```

## Arguments

- x:

  sf. or any object of a class that can be read by sf::st_crs such as
  "LAS" class

## Value

A data.frame with the units and scale factor for the vertical projection
