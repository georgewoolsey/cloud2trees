# Package index

## All functions

- [`chunk_las_catalog()`](https://georgewoolsey.github.io/cloud2trees/reference/chunk_las_catalog.md)
  :

  Tile raw `.las`\|`.laz` files to work with smaller chunks

- [`cloud2raster()`](https://georgewoolsey.github.io/cloud2trees/reference/cloud2raster.md)
  : Use raw .las\|.laz files to generate CHM, DTM, and Normalized .las
  files

- [`cloud2trees()`](https://georgewoolsey.github.io/cloud2trees/reference/cloud2trees.md)
  : Use raw .las\|.laz files to generate CHM, DTM, and a tree list

- [`cloud2trees_to_lanl_trees()`](https://georgewoolsey.github.io/cloud2trees/reference/cloud2trees_to_lanl_trees.md)
  :

  Use outputs from
  [`cloud2trees()`](https://georgewoolsey.github.io/cloud2trees/reference/cloud2trees.md)
  to generate inputs for LANL TREES program

- [`create_lax_for_tiles()`](https://georgewoolsey.github.io/cloud2trees/reference/create_lax_for_tiles.md)
  :

  Create spatial index `.lax` files

- [`find_ext_data()`](https://georgewoolsey.github.io/cloud2trees/reference/find_ext_data.md)
  : Find the location of external data

- [`get_data()`](https://georgewoolsey.github.io/cloud2trees/reference/get_data.md)
  : Download all external data used by package

- [`get_foresttype()`](https://georgewoolsey.github.io/cloud2trees/reference/get_foresttype.md)
  : Download Forest Type Groups of the Continental United States data

- [`get_landfire()`](https://georgewoolsey.github.io/cloud2trees/reference/get_landfire.md)
  : Download Forest Type Groups of the Continental United States data

- [`get_treemap()`](https://georgewoolsey.github.io/cloud2trees/reference/get_treemap.md)
  : Download TreeMap 2016 data

- [`itd_tuning()`](https://georgewoolsey.github.io/cloud2trees/reference/itd_tuning.md)
  : Individual Tree Detection (ITD) tuning

- [`itd_ws_functions()`](https://georgewoolsey.github.io/cloud2trees/reference/itd_ws_functions.md)
  : Individual Tree Detection (ITD) functions

- [`ladderfuelsr_cbh()`](https://georgewoolsey.github.io/cloud2trees/reference/ladderfuelsr_cbh.md)
  :

  estimate CBH for a *single tree* using `LadderFuelsR` package

- [`leafr_for_ladderfuelsr()`](https://georgewoolsey.github.io/cloud2trees/reference/leafr_for_ladderfuelsr.md)
  :

  re-writes `leafR` steps to allow for treeID as input for
  `ladderfuelsR` or
  [`ladderfuelsr_cbh()`](https://georgewoolsey.github.io/cloud2trees/reference/ladderfuelsr_cbh.md)

- [`polygon_attribute_to_las()`](https://georgewoolsey.github.io/cloud2trees/reference/polygon_attribute_to_las.md)
  : function to attach polygon attribute to point cloud

- [`raster2trees()`](https://georgewoolsey.github.io/cloud2trees/reference/raster2trees.md)
  : Use a CHM raster to detect individual trees

- [`simplify_multipolygon_crowns()`](https://georgewoolsey.github.io/cloud2trees/reference/simplify_multipolygon_crowns.md)
  :

  Simplify MULTIPOLYGON to POLYGON geometry in an `sf` class object

- [`treels_stem_dbh()`](https://georgewoolsey.github.io/cloud2trees/reference/treels_stem_dbh.md)
  :

  detect tree stems and estimate DBH using `TreeLS` package

- [`trees_biomass()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_biomass.md)
  : Estimate tree biomass (or crown biomass) for a tree list

- [`trees_biomass_cruz()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_biomass_cruz.md)
  : Estimate tree crown biomass for a tree list based on Cruz et al.
  (2003)

- [`trees_biomass_landfire()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_biomass_landfire.md)
  : Estimate tree crown biomass for a tree list using LANDFIRE data

- [`trees_cbh()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_cbh.md)
  : Estimate CBH using tree crown polygons and normalized point cloud
  data

- [`trees_competition()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_competition.md)
  : Calculate competition metrics for a tree list

- [`trees_dbh()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_dbh.md)
  : Estimate DBH for a tree list based on height

- [`trees_hmd()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_hmd.md)
  : Estimate HMD using tree crown polygons and normalized point cloud
  data

- [`trees_landfire_cbd()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_landfire_cbd.md)
  : Extract LANDFIRE CBD raster cell value for a tree list based on
  location

- [`trees_type()`](https://georgewoolsey.github.io/cloud2trees/reference/trees_type.md)
  : Estimate forest type for a tree list based on location
