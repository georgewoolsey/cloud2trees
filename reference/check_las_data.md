# Check a for a directory with .laz\|.las files, the path of a file, or an object of class `LAS`\|`LASCatalog`

Check a character for a directory with .laz\|.las files, the path of a
file, or an object of class `LAS`\|`LASCatalog`

## Usage

``` r
check_las_data(las)
```

## Arguments

- las:

  character. a directory with .laz\|.las files files, the path of a
  single .laz\|.las file, -or- an object of class `LAS`\|`LASCatalog`
